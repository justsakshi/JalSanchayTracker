# Jal-Sanchay Tracker

A rainwater harvesting calculator and tracker for Android.

Enter your roof area, rainfall, and tank capacity — the app calculates how much water you can collect, how many days it'll last, and saves every entry so you can track your impact over time.

---

## Features

- Calculates rainwater collected based on roof area and rainfall
- Animated tank fill visualization
- Daily and monthly reports with a bar chart
- All-time and today's savings dashboard
- Water coverage drop grid
- Tips for rainwater harvesting

## Tech Stack

- Kotlin + Android SDK
- Room (local database)
- Coroutines
- Custom Canvas chart view (no third-party chart library)

## Getting Started

Clone the repo and open in Android Studio. Build and run on any Android device or emulator running API 26+.

```bash
git clone https://github.com/justsakshi/JalSanchayTracker.git
```
